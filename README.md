# ArchitectureInfo
这是提供给Rocket Unturned Server的一个插件。

## 功能
查看玩家建筑物的所有者（SteamID以及Steam账号页）  

## 使用方法
视线光标对准待查询建筑物，并使用/archi指令
若查询到建筑物所有人，使用/spconfirm指令让服务端请求客户端开启游戏内网页，显示建筑物所有人的Steam首页。

## 权限节点
archinfo: 使用/archi命令以及/spconfirm命令的权限。
